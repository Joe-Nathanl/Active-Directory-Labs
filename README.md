# Active Directory Labs

## Overview

This repository documents my hands-on Active Directory lab work as I continue developing practical system administration skills.

The purpose of these labs is to gain experience with core Active Directory administration tasks commonly performed by Help Desk Technicians, System Administrators, and Network Administrators in enterprise Windows environments.

---

## Lab Environment

### Technologies Used

- Windows Server
- Active Directory Domain Services (AD DS)
- Windows Client Workstations
- DNS
- Group Policy
- Virtual Machines

---

## Skills Demonstrated

### User Account Administration

- Creating user accounts
- Modifying user properties
- Disabling and enabling accounts
- Unlocking locked accounts
- Performing password resets

### Group Management

- Creating security groups
- Managing group membership
- Assigning users to appropriate groups
- Understanding role-based access control

### Organizational Unit (OU) Management

- Creating Organizational Units
- Moving users between OUs
- Structuring directory objects
- Organizing administrative boundaries

### Computer Account Administration

- Managing computer accounts
- Understanding domain-joined devices
- Moving computer objects between OUs

### Active Directory Fundamentals

- Domain concepts
- Authentication and authorization
- User and group relationships
- Directory object management

### Group Policy Fundamentals

- Understanding Group Policy Objects (GPOs)
- Applying policies to Organizational Units
- Managing centralized configuration settings

## Lab Exercises

### Active Directory Basics

- Installed and configured Active Directory Domain Services
- Created a domain environment
- Explored Active Directory Users and Computers (ADUC)
- Navigated directory structures and administrative tools

### User Management Lab

Created multiple user accounts and practiced:

- Password resets
- Account lockout recovery
- Account disable/enable procedures
- User property management

### Security Groups Lab

Configured security groups and managed:

- Group membership assignments
- Access control concepts
- Department-based group organization

### Organizational Units Lab

Created Organizational Units for:

- IT
- HR
- Quality Assurance
- Workstations

Practiced moving users and computers into appropriate containers.

### Group Policy Lab

Studied Group Policy administration by:

- Creating GPOs
- Linking policies to OUs
- Testing policy application
- Understanding centralized administration
